# QuizMaster

A fast-paced, interactive web-based trivia application built with vanilla JavaScript. Test your knowledge across multiple categories against a 20-second timer!

## 🚀 Features

*   **Multiple Categories:** Choose from Technology, Cybersecurity, and General Knowledge.
*   **Dynamic Quizzes:** A randomized set of 5 questions is selected from the pool for each session.
*   **Time Attack:** A built-in 20-second countdown timer for each question to keep you on your toes.
*   **Real-time Feedback:** Get instant validation on your answers and track your score throughout the quiz.
*   **Results Summary:** View your final score with personalized feedback messages based on your performance.
*   **Modern UI:** A clean, responsive design using modern fonts and styling for a great user experience.

## 🛠️ Technologies Used

*   **HTML5:** Semantic structure of the application.
*   **CSS3:** Custom styling, animations, and responsive layout without the use of external frameworks.
*   **Vanilla JavaScript:** Core application logic, DOM manipulation, state management, and timers.

## 📂 Project Structure

*   `quiz_app.html`: The main HTML document containing the structure for the Home, Quiz, and Result screens.
*   `styles.css`: The stylesheet handling the layout, typography, interactive states, and transitions.
*   `script.js`: The JavaScript file containing the quiz data, logic for rendering questions, checking answers, timer functionality, and score calculation.

## 💻 How to Run Locally

1.  Clone this repository to your local machine.
2.  Open the `quiz_app.html` file in your preferred web browser. No local server or build tools are required!
